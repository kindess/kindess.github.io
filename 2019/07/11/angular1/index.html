Angular入门学习一

  • Angular
  • node

##安装环境

###安装node.js
学习Angular，首先需要安装node.js，需要node内置的npm支持，node.js的官网下载

###安装angular
使用node安装angular：
npm的官网在国外，同docker等等，网速太慢会引起各种问题，因此使用node安装之前，先换源，使用淘宝镜像安装。

npm config set registry https://registry.npm.taobao.org //淘宝换源指令
npm install -g @angular/cli //安装angular

查看版本信息，是否安装成功，运行命令 ng v

新建一个angular空项目不过，使用上面这条命令创建会出现问题（没遇到就算了，遇到请使用下面的方式创建）

ng new --skip-install

如图，

上面我们没有安装angular的依赖，因此我们进入项目根目录下，我这个项目根目录是C:\Users\Administrator\my-app,
打开cmd命令，执行npm install或者cnpm install（上面换源了才能使用cnpm命令）
运行命令

ng serve

打开浏览器，访问localhost:4200，如果执行命令出现类似下面的报错，

An unhandled exception occurred: Cannot find module '@angular-devkit/build-webpack'
See "C:\Users\ADMINI~1\AppData\Local\Temp\ng-lEA4YS\angular-errors.log" for further details.
npm ERR! file C:\Windows\system32\cmd.exe
npm ERR! code ELIFECYCLE
npm ERR! errno ENOENT
npm ERR! syscall spawn
npm ERR! test@0.0.0 start: ng serve
npm ERR! spawn ENOENT
npm ERR!

npm ERR! D:\Program Files\nodejs\node_cache_logs\2019-07-11T06_13_09_014Z-debug.log

表示缺少依赖，最直接的办法就是删掉my-app项目中的文件夹node_modules，这文件夹下全是执行cnpm install 命令安装的依赖，删掉之后我们重新执行cnpm install进行重新下载。如仍有报错，继续。。。 ###结果 ![成功页面](angular1/3.png) ###新建一个angular空项目 ng new 项目名
